## How to Upgrade

Follow these steps to update Hexus and access the latest features.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run this command in your terminal:
    
    ```bash
    hexus --version
    ```
  </Step>
  <Step title="Update CLI" icon="download">
    <CodeGroup tabs="npm,yarn">
      ```bash
      npm install -g @hexus/cli@latest
      ```
      ```bash
      yarn global add @hexus/cli@latest
      ```
    </CodeGroup>
  </Step>
  <Step title="Restart Services" icon="refresh-cw">
    Restart your Hexus dashboard and verify changes.
    
    ```bash
    hexus restart
    ```
  </Step>
</Steps>

<Callout kind="tip">
  Enable auto-updates in your account settings to receive new versions without manual intervention. Set `autoUpdate: true` in your config file.
</Callout>
